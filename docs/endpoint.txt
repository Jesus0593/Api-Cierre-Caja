USUARIOS
    GET /api/usuarios/getUsuarios?pass
        Retorna la informacion de un usario en especifico

EMPRESAS
    GET /api/empresas/getEmpresas?id
        Retorna la informacion de todas las empresas que puede manejar un usuario

    GET /api/empresas/getTiendasCajas?database
        Retorna la informacion de todas las tiendas de una empresas
        *database tipo gestion

CUENTAS
    GET /api/cuentas/getConfiguracion
        Retorna las cuentas para manejo de Faltantes y Sobrantes
    
    GET /api/cuentas/getAllCuentas?database
        Retorna Todas las cuentas para poder elegir las cuentas de Faltantes y Sobrantes
        *database de tipo contable B190968E988D97918B7E8186

    POST 
        Actualiza las cuentas de Faltantes y Sobrantes
        * requiere un json const { FALTANTEVENTA,SOBRANTEVENTA,FALTANTEREDONDEO,SOBRANTEREDONDEO }

MONEDAS 
    GET /api/monedas/getConvertir?database?fecha?importe
        Realiza las diferentes conversiones entre las diferentes monedas que posea la tienda
        *database de tipo gestion 8F9393A4829897
